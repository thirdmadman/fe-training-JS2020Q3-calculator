Expression calculator
This calculator performs calculations on the fly, which means that technically there is no need for an equals button, but it is still there. This calculator can work with fractional numbers and negative ones too This calculator does not include handling all the exceptions that may arise during a variety of mathematical operations. The root function works quite simply, it takes as the first operand the base of the root, which will be used for calculation. As the second operand, you enter the number from which you want to extract the root. The function n ^ i works as simply as it looks, it takes the number n and raises it to the i power. Calculator understands sequential input of operations. You can enter 1 + 1 first, then press -.
